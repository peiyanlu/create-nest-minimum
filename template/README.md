## Compile and run the project

```bash
# development
$ $RUN build

# watch mode
$ $RUN start:dev

# debug mode
$ $RUN start:debug

# production mode
$ $RUN start:prod
```

## Run tests

```bash
# unit tests
$ $RUN test

# e2e tests
$ $RUN test:e2e

# test coverage
$ $RUN test:cov
```
